# Avalanche-Rose-Easel
An Interactive avalanche elevation rose in 3D with interactive elevation marker bands and customizeable hazards.

## How to use:

1: Head to the link for the github pages page: <a href='https://mat-foucher.github.io/Avalanche-Rose-Easel/'>https://mat-foucher.github.io/Avalanche-Rose-Easel/</a>

2: Double click on the cobweb to cycle through hazard severity, or triple click + drag to spread hazard level across multiple sections:

<img width="1898" height="868" alt="image" src="https://github.com/user-attachments/assets/189e07dd-a52d-4331-af53-cf8a08c12f76" />

3: Click + drag on cobweb diagram to reveal 3D profile + elevation marker bands: 

<img width="1909" height="774" alt="image" src="https://github.com/user-attachments/assets/1d83c7a5-b9ca-4344-bc7a-57ded16b9113" />

4: Drag Elevation sliders to change elevation bands on the cobweb: 

<img width="1907" height="790" alt="image" src="https://github.com/user-attachments/assets/eaa6f172-3ade-4a76-abda-b36970a3f8e6" />
